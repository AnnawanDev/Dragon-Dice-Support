# Dragon Dice — Support & Feedback

Thank you for using Dragon Dice! We want to ensure you have the best possible experience rolling your dice and tracking your characters. 

If you are encountering an issue, have a feature request, or need assistance, please use one of the methods below to get in touch.

---

## How to Get Help

### 1. Report an Issue or Feature Request
The fastest way to get support is to open an issue directly on our project tracker. This allows you to see if other users are experiencing the same issue and track the progress of a fix.

*   [**Open a Support Ticket / Bug Report**](https://github.com/AnnawanDev/Dragon-Dice-Support/issues)

*Note: You will need a free GitHub account to open an issue here.*

## Frequently Asked Questions (FAQ)

**Q: Will there be more themed dice styles added in the future?**  
**A:** Yes! We are working on new character-themed animations. Feel free to submit a feature request via the links above if there is a specific style you want to see.
